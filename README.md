# 🚀 AI Interview System

An AI-powered platform that generates personalized interview strategies, technical questions, and skill gap analysis based on job descriptions and user profiles.

---

## 🔥 Features

- 🧠 AI-generated interview questions
- 📄 Resume-based evaluation
- 📊 Skill gap analysis
- 📥 Downloadable PDF reports
- 🔐 JWT Authentication
- 📁 File upload (Resume)

---

## 🛠 Tech Stack

### Frontend
- React (Vite)
- Tailwind / SCSS

### Backend
- Node.js
- Express.js
- MongoDB

### Other
- Google GenAI API
- Puppeteer (PDF generation)
